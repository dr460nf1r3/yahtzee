Class: YahtzeeGame

YahtzeeGame(…names)

Class representing a Yahtzee game with multiple players.

Constructor

new YahtzeeGame(…names)

Create a Yahtzee game with a number of players. It allows creating a game with multiple players, but the current implementation hardcodes it one player.
Parameters:
